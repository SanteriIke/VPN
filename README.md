VPN
Formål
En simple made at download VPN-Pakker til ens computer, man kan benytte sig af hvis man normalt har hjemmearbejde. 
Funktioner
Der er pakker til 3 enheder lige nu. Windows, iOS og Android. Lige nu er der ikke en arm64 pakke, men der kommer snart en. Der er guides til hvordan man installere pakkerne hvis det er man er i tvivl. Pakkerne kan også opdateres ved at logge ind med admin koden.

Purpose
To provide a simple method for downloading VPN-Packages for ones computer, that can be used by those who regularly work from at home and need access to the domain.
Features
There are packages for 3 devices right now. Windows, iOS and Android. Currently there is no arm64 package, but it will be available soon enough. There is a guide for installing the packages. Packages can also be updated by entering the admin code.
